# 使う前に
1. drawioをインストールする
    1. aptパッケージを更新する
    ```
    apt update && apt upgrade
    ```
    2. パッケージをダウンロードする
    ```
    wget https://github.com/jgraph/drawio-desktop/releases/download/v22.1.2/drawio-amd64-22.1.2.deb
    ```
    3. インストールする
    ```
    apt install ./drawio-arm64-22.1.2.deb
    ```
